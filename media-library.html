Genesis

This repository was created as a place to store images that I use in other GitHub repositories, but which I do not have permission to distribute.

Purpose

By storing licensed media in this separate repository, and not in the repositories that use them, I am meeting the requirements of licenses which allow me to use images and other media, but do not allow me to distribute them when others clone the repositories.
