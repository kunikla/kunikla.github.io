Kunikla

StreamOnly Omeka Plugin

Genesis

This project is being developed under the auspices of Code for BTV, a brigade of Code for America, for the benefit of the Vermont Music Archive, a project of the non-profit Big Heavy World.

Purpose

The goal is to create a plugin for the Omeka Classic CMS that will allow the Vermont Music Archive website and other Omeka-based websites to stream audio files, but not allow visitors to download the files. Big Heavy World and other organizations often have licenses permitting them to possess some number of copies of each audio file, but not to distribute copies of the files. The number of visitors who can simultaneously stream an audio file is configurable on an Item-by-Item basis, where an Item is assumed to be a group of audio files associated with a single artist. The amount of time an audio file is reserved for streaming is also configurable on an Item-by-Item basis.

It is always possible for DRM protections to be circumvented, but the most common methods of downloading audio files are blocked. The goal is to keep the typical visitor honest.
