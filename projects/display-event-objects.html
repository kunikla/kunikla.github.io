Kunikla

Display Event Objects

Genesis

This app was created in response to a suggestion by Zac Gordon in his course JavaScript for WordPress, in Lesson 1.4.8 - A Look at More Events.

Purpose

I and other students of the JavaScript for WP course can use this app to familiarize ourselves with the Event Objects of commonly used events.

Credits

The code borrows heavily from the sample code for Lesson 1.4.1 - Introduction to DOM Events. Some event handlers were added, and the choice of which event object to display is facilitated by my menu system.
