Kunikla

Big Picture Omeka Classic Theme, customized

Genesis

This project is being developed under the auspices of Code for BTV, a brigade of Code for America, for the benefit of the Vermont Music Archive, a project of the non-profit Big Heavy World.

Purpose

The creators of the Vermont Music Archive website have selected the Omeka CMS for their platform, and the Big Picture theme for the layouts. Each album will be stored as an Item, and when that Item is selected, a high-resolution image of the album cover is displayed prominently on the left, and a list of songs on the right, along with information about the album.

Due to browser and server limitations, websites that use the StreamOnly plugin can only display a limited number of HTML5 audio elements for protected audio files on each page. The latest version of the Big Picture theme, currently version 1.2, was modified so that when a single Item/album is displayed, only one HTML5 audio element is displayed on that page. If more than one audio file is associated with that Item/album, a form with a Select field is displayed, to allow the site visitor to select which video is available to be streamed.
